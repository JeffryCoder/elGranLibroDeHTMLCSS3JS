1.1 Componentes básicos

  HTML da la estructura de el sitio web
  CSS da la apariencia de la escructura a presentar en pantalla
  JavaScript da la funcionalidad e interactividad necesaria al sitio web

1.2 Estructura global

  <!DOCTYPE html> Etiqueta para declarar el tipo de documento que estamos haciendo

  <html> Etiqueta que comienza a definir la escructura del sitio web
  <HEAD> Este etiqueta define la cabecera del documento. TODO lo que se ponga aquí excepto los títulos y algunas íconos será invisible para el usuario
  <meta> Especifica de qué manera el texto será presentado en pantalla
    <link> Se utiliza para cargar archivos CSS externos
